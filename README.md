# 🚨 LifeSaver AI

LifeSaver AI is an AI-powered emergency response assistant that provides instant first-aid guidance.

## 🔥 Features
- AI-based emergency detection
- Voice input support 🎤
- Nearby hospital locator 📍
- Clean and responsive UI

## 🛠 Tech Stack
- HTML, CSS, JavaScript
- Python (Flask)
- Machine Learning (Scikit-learn)

## 🚀 How to Run

### Backend
cd backend  
python app.py  

### Frontend
Open frontend/index.html in browser  

## 🎤 Demo
Try:
"person not breathing"

## 🌍 Impact
Helps people respond quickly in emergencies and potentially save lives.
